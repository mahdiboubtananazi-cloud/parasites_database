# Digital Parasites Database - User Guide

## Website Purpose

This website serves as a comprehensive digital archive for parasites discovered in the Parasitology Department. It allows students, researchers, and faculty to browse, search, and contribute to a growing collection of parasite specimens with detailed scientific information and microscopic images.

## Using Your Website

### 1. Browse the Parasite Collection

Start by clicking **"الطفيليات"** (Parasites) in the navigation menu. You'll see a grid of parasite cards, each displaying:
- A microscopic image of the parasite
- Scientific name (in italics)
- Arabic or French name (depending on your language selection)
- Host species information
- Year of discovery

### 2. Search and Filter Parasites

On the **Parasites** page, use the search bar to find parasites by:
- Scientific name (e.g., "Plasmodium falciparum")
- Arabic name (e.g., "البلازموديوم")
- French name (e.g., "Giardia lamblia")

Use the filter dropdowns to narrow results by:
- **Host Species**: Filter parasites by the organism they infect
- **Discovery Year**: Find parasites discovered in specific years

Click **"Reset Filters"** to clear all filters and start over.

### 3. View Detailed Information

Click on any parasite card to see comprehensive details including:
- Full scientific name and common names in multiple languages
- **Description**: Scientific overview of the parasite
- **Morphological Characteristics**: Physical features visible under microscope
- **Detection Method**: Laboratory techniques used to identify the parasite
- **Host Species**: The organism it infects
- **Discovery Year**: When it was first documented
- **Microscopic Images**: High-quality photos for visual reference

### 4. Add New Parasites (Login Required)

To contribute new parasite records:
1. Click **"دخول"** (Login) in the top menu
2. If you don't have an account, click **"تسجيل"** (Register) and create one
3. Once logged in, click **"إضافة طفيلي"** (Add Parasite)
4. Fill in the form with:
   - Scientific name (required)
   - Common name and translations
   - Host species information
   - Morphological characteristics
   - Detection methods
   - Upload a microscopic image
5. Click **"إرسال"** (Submit) to save

### 5. Add Sample Records (Login Required)

To document collected samples:
1. Log in to your account
2. Click **"إضافة عينة"** (Add Sample)
3. Select the parasite from the dropdown
4. Enter sample details:
   - Unique sample number (e.g., S-2024-001)
   - Host specimen information
   - Collection date and location
   - Additional notes
5. Click **"إرسال"** (Submit)

### 6. Change Language

The website supports three languages. Switch between them using the language buttons in the top navigation:
- **العربية** (Arabic) - Default
- **Français** (French)
- **English**

Your language preference is maintained as you browse.

## Managing Your Website

Click the **Settings** icon in the Management UI (right panel) to:

- Manage user notifications

- Update environment variables and secrets

Use the **Database** panel in the Management UI to:
- Browse all parasites and samples
- Edit existing records
- Delete outdated entries
- Export data if needed

- Page views and unique visitors
